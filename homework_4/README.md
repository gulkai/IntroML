# Деревья. Случайный лес
## Описание библиотек
numpy, pandas - обработка данных

matplotlib, seaborn - визуализация

scikit-learn - модели, метрики, кодирование

стандартные (collections, itertools, typing)


## Ход работы
Деревья (часть 1):

Реализованы формулы энтропии, индекса Джини, критерия информативности

Создана функция find_best_split для поиска оптимального порога

Протестирована на датасете California Housing

Деревья (часть 2):

Реализован класс DecisionTree для классификации и регрессии

Протестирован на mushrooms (accuracy ≈ 100%)

Ансамбли (часть 3):

Сравнение трех моделей на diabetes датасете:

Одиночное дерево: accuracy 74.5%

Bagging (50 деревьев): accuracy 78.8%

Random Forest (50 деревьев): accuracy 78.8%

Подбор гиперпараметров через GridSearchCV

Анализ важности признаков (Glucose - самый важный, 32%)

Исследование зависимости качества от числа деревьев

##ВЫВОДЫ:

Ансамблевые методы (Bagging, Random Forest) стабильно лучше одиночных деревьев

Оптимальное число деревьев: 50-100 (дальнейший рост не дает улучшений)

Самый важный признак для диабета - уровень глюкозы (медицински обоснованно)

Random Forest показывает лучший AUC-ROC (0.837)
